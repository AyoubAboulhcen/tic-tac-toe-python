# Tic-Tac-Toe Python 

After finishing the Python Essentials 1 course by Cisco/OpenEDG, I needed
to actually test what I learned. So I sat in a café for 5 hours and built this.

## What is it
A command line Tic-Tac-Toe game built from scratch in Python. This is the
final project of the Python Essentials 1 course. The computer plays X and
always starts in the middle square. Good luck beating it.

## The Honest Part
The hardest part was adapting to Cisco's specific requirements. I could have
written simpler code my own way, but following the constraints taught me more
than I expected.

## What I learned
- How to structure a program using functions
- How to work with 2D lists accessed via board [row],[column]
- Input validation using try/except
- Game logic and win detection across 8 combinations
- Random computer moves using Python's random module

## How to run
Make sure Python is installed, then run:
python tic_tac_toe.py

## How to play
- Computer places X in the middle automatically
- Enter a number from 1 to 9 to place your O
- Numbers on the board show which squares are free
- First to get 3 in a row wins
